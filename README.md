# Gestionare-Biblioteca-Digitala
Prototipul + Descriere totala in Word despre o aplicație desktop " Gestionarea Bibliotecii Digitale "

Gestionare Bibliotecă Digitală este o aplicație desktop care permite organizarea și administrarea unei colecții de cărți digitale. Utilizatorii pot căuta și vizualiza cărți din bibliotecă, iar administratorul poate adăuga și edita informațiile despre cărți printr-o interfață simplă și intuitivă.
